# Miyabi Calculator Tools

miyabi-mcp-bundleのCalculator Tools カテゴリ（3ツール）をClaude Codeから活用するスキルです。

## 利用可能なMCPツール

| Tool | 説明 |
|------|------|
| `calc_expression` | 数式の安全な評価 |
| `calc_unit_convert` | 単位間の変換 |
| `calc_statistics` | 統計量の計算（平均・中央値・標準偏差等） |

## 使い方の例

- 「1024 * 768 の計算結果は？」→ `calc_expression`
- 「100MBは何GBか？」→ `calc_unit_convert`
- 「[85, 92, 78, 95, 88] の平均と標準偏差は？」→ `calc_statistics`
